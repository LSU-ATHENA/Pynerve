# References

- PH6 tracks active TDA research. See the Pynerve release notes for specific algorithms included in each version.
- Bauer, U. (2021). Ripser: Efficient Computation of Vietoris-Rips Persistence Barcodes. *Journal of Applied and Computational Topology*, 5(3), 391-423.
- De Silva, V., Morozov, D., Vejdemo-Johansson, M. (2011). Dualities in Persistent Cohomology. *Inverse Problems*, 27(12), 124003.
- Chen, C., Kerber, M. (2011). Persistent Homology Computation with a Twist. *Proc. 27th European Workshop on Computational Geometry*.
- The adaptive pivoting strategy is inspired by techniques in sparse matrix computation (Davis, 2006, *Direct Methods for Sparse Linear Systems*).
- The block-sparse reduction approach follows cache-blocking strategies common in high-performance computing (Lam, Rothberg, Wolf, 1991, *The Cache Performance and Optimizations of Blocked Algorithms*).
- Speculative reduction is a novel approach inspired by speculative execution in modern CPU architectures (Hennessy & Patterson, 2019, *Computer Architecture: A Quantitative Approach*).
- Otter, N., Porter, M.A., Tillmann, U., Grindrod, P., Harrington, H.A. (2017). A Roadmap for the Computation of Persistent Homology. *EPJ Data Science*, 6(1), 17.
