# PH4 Engine

> Engine: `PersistenceEngine.PH4`. Cohomology with dimension-cascading clearing. Recommended for datasets with <100K points and max_dim <= 4.

PH4 is a fast, battle-tested persistent homology engine for Z2 coefficients that combines standard and cohomology reduction with clearing, compression, and approximate modes.
