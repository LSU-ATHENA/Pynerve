# When to Use

- **Validated pipelines** requiring checksums and stability certificates
- **Regulated environments** with strict reproducibility mandates
- **Distributed systems** needing result verification across nodes
- **CI/CD** where computation correctness must be verified programmatically
- **Gradient-based learning** via differentiable PH5 ops
- **When extended clearing is beneficial** (large, moderately dense filtrations)
